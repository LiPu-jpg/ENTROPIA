# 研究归档

本目录是经过整理的历史探索材料归档。

原始工作区包含多个研究轮次、方向选择笔记、候选工作区和草稿报告。这些原始中间文件已被合并整理为规模更小的规范化文档集合。目标是保留决策链，同时去除重复或部分重叠的草稿。

## 来源

本归档合并了以下来源的信息：

- 从 git 中可恢复的旧版 ENTROPIA v1 算法、扩展和论文笔记文档；
- 整合后的 v2 设计文档：`../design_v2_reward_routing.md`、`../paper_formula_methodology.md`、`../related_work.md`；
- `../../../references/papers/` 下保留的原始论文 PDF/文本；
- 清理后的研究工作区中的方向选择结论。

由于当前文件系统状态无法精确还原已删除的未跟踪研究文件夹，因此本归档保留的是提炼后的决策，而非原始草稿快照。
