# 大模型RL顶会论文深度研究计划

## 目标
调研2024-2026年顶会中关于大模型（LLM）强化学习（RL）的论文，重点关注：
1. 使用的Benchmark数据集
2. 采用的方法（算法/框架）
3. 具体实现细节

## 研究维度
- **会议范围**: NeurIPS, ICML, ICLR, ACL, EMNLP, AAAI, WWW, SIGIR等
- **时间范围**: 2024-2026年
- **主题**: LLM + RL（包括RLHF、DPO、PPO、在线/离线RL、多轮RL等）
- **研究角度**: Benchmark, Methodology, Implementation

## 阶段规划

### Stage 1 — 多Agent并行探索（Phase 1 & 2 of deep-research-swarm）
- 部署多个Research Agent并行搜索不同子领域
- Agent 1: 搜索RLHF相关论文（PPO, DPO, IPO等）
- Agent 2: 搜索在线RL与离线RL在LLM中的应用
- Agent 3: 搜索多轮/迭代RL、工具使用RL等新兴方向
- Agent 4: 搜索奖励模型与评估Benchmark

### Stage 2 — 深度调研与交叉验证
- 对Stage 1发现的论文进行深度调研
- 提取每篇论文的benchmark、方法、实现细节
- 交叉验证，补充缺失信息

### Stage 3 — 综合分析与报告撰写
- 汇总所有发现，按类别组织
- 撰写完整研究报告
- 输出docx格式
